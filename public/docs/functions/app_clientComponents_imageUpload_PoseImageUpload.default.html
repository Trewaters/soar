      A React component for uploading images associated with a specific asana pose.

      Supports both "button" and "dropzone" variants for file selection, drag-and-drop, file type and size validation, image preview, alt text input for accessibility, and error handling. Integrates with user session and associates the uploaded image with a pose by ID and name.

      Parameters

      • props: PoseImageUploadProps

        Component props.

        Props for the PoseImageUpload component.

        • OptionalacceptedTypes?: string[]

          Optional array of accepted MIME types for image uploads (e.g., ['image/jpeg', 'image/png']).

        • OptionaliconSize?: "large" | "medium" | "small"
        • OptionalmaxFileSize?: number

          Optional maximum file size allowed for uploads, in megabytes (MB).

        • OptionalonImageDeleted?: (imageId: string) => void

          Optional callback invoked when an image is deleted. Receives the ID of the deleted image.

        • OptionalonImageUploaded?: (image: PoseImageData) => void

          Optional callback invoked when an image is successfully uploaded. Receives the uploaded image data as a parameter.

        • OptionalposeId?: string

          Optional identifier for the pose associated with the image upload.

        • OptionalposeName?: string

          Optional name of the pose associated with the image upload.

        • OptionalrefreshTrigger?: any
        • OptionalshowSlotInfo?: boolean

          Optional flag to show remaining upload slots information.

        • Optionalvariant?: "button" | "dropzone" | "icon-button"

          Optional UI variant for the upload component. Can be either 'button' or 'dropzone'.

      Returns Element

      The image upload UI.

      Component

      Example

      <PoseImageUpload
        poseId="123"
        poseName="Downward Dog"
        onImageUploaded={(img) => console.log(img)}
        variant="dropzone"
      />
