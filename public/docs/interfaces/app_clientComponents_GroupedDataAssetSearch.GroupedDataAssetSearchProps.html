    Properties

    displayField

    displayField: (item: T) => string

    Function to extract the text used for display in dropdown options. In most cases this is the same as searchField.

    Example

    (item) => item.seriesName

    OptionalfullWidth

    fullWidth?: boolean

    When true, stretches the search container to full available width instead of the default centered max width behavior.

    Default

    false

    getCreatedBy

    getCreatedBy: (item: T) => string | undefined

    Function to extract the creator identifier from an item. Used to determine which group the item belongs to.

    Example

    (item) => item.createdBy

    OptionalincludeOthersGroup

    includeOthersGroup?: boolean

    If true, a third "Others" section is rendered for items created by users who are neither the current user nor alpha users.

    Default

    false

    OptionalinputValue

    inputValue?: string

    Controlled input value for the search field. When provided the component operates in controlled mode.

    items

    items: T[]

    Pre-fetched array of items to search through. The component handles grouping and sorting internally.

    Optionalloading

    loading?: boolean

    If true, shows a loading indicator inside the Autocomplete.

    Default

    false

    myLabel

    myLabel: string

    Label for the section of items created by the current user.

    Example

    "My Flows"

    OptionalonClose

    onClose?: () => void

    Callback fired when the Autocomplete closes.

    OptionalonInputChange

    onInputChange?: (value: string) => void

    Callback fired when the search input value changes (controlled mode).

    OptionalonOpen

    onOpen?: () => void

    Callback fired when the Autocomplete opens.

    onSelect

    onSelect: (item: T) => void

    Callback fired when the user selects an item from the dropdown. Section headers are never passed to this callback.

    Optionalopen

    open?: boolean

    Whether the Autocomplete dropdown is open (controlled mode).

    OptionalothersLabel

    othersLabel?: string

    Optional label for the "Others" group (used only when includeOthersGroup is true).

    Default

    "Others"

    OptionalplaceholderText

    placeholderText?: string

    Placeholder text shown inside the search input.

    Default

    "Search..."

    publicLabel

    publicLabel: string

    Label for the section of items created by alpha/public users.

    Example

    "Public Flows"

    searchField

    searchField: (item: T) => string

    Function to extract the text field used for search/filtering.

    Example

    (item) => item.seriesName
