# Practice Asanas Help

Search and browse through your available yoga poses. Use the search bar to find specific asanas or scroll through the complete collection to explore and practice different poses.

## How to Use This Page

1. **Search for poses** - Type the English name in the search box
2. **Browse results** - Results are organized by:
   - **My Asanas** - Poses you created
   - **Public Asanas** - Community poses
   - **Others** - All other poses
3. **Select a pose** - Click to view detailed information

## Tips

- Partial matches work (e.g., "war" shows "Warrior I", "Warrior II")
- Page auto-refreshes when you return from creating a pose
- Use the back button to return to Asana Landing Page
