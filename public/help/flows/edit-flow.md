# Edit Flow Help

## Overview

The **Edit Flow** page allows you to modify existing yoga flow sequences by updating the flow name, adding or removing asanas, reordering poses, customizing alignment cues, managing images, and changing the flow description. Edit your flows to refine your practice sequences as your yoga journey evolves.

**Page URL:** `/navigator/flows/practiceSeries?id={flowId}` (Edit mode accessed via the Edit icon)

## What Can You Edit?

When you own a flow (or have admin permissions), you can modify:

- **Flow Name** - Change the title and identifier
- **Flow Description** - Update practice notes and purpose
- **Difficulty Level** - Adjust beginner/intermediate/advanced classification
- **Asana List** - Add, remove, or reorder poses in the sequence
- **Alignment Cues** - Add detailed instructions for each pose (up to 1000 characters)
- **Flow Images** - Upload, replace, or remove visual references

**Note:** You can only edit flows that you created or flows you have admin permissions for. The edit button (pencil icon) will only appear if you have edit access.

## Page Components

### Flow Title with Edit Button

At the top of the flow detail page, the flow name is displayed as a large heading. For flows you own, a small **Edit** icon (pencil) appears next to the title. Click this icon to enter edit mode and modify the flow inline.

### Edit Mode Interface

When you click the Edit icon, the flow detail view transforms into a comprehensive editor with the following sections:

#### Details Section

**Flow Name (Required)**

- Text field containing the flow's current name
- Must be unique and descriptive
- Changes reflect immediately when saved
- Examples: "Morning Energy Flow", "Core Power Builder"

**Description**

- Multi-line text area for detailed flow explanation
- Include the flow's purpose, benefits, and recommendations
- Supports line breaks for formatting
- Optional but highly recommended

**Difficulty Level**

- Dropdown selector with three options:
  - Beginner - Gentle introduction to yoga
  - Intermediate - Moderate challenge and complexity
  - Advanced - Demanding poses requiring experience
- Helps practitioners choose appropriate flows

**Created By**

- Read-only field showing the flow creator's identifier
- Displayed for reference but cannot be modified
- Helps track flow ownership

#### Flow Image Manager

**Upload New Images**

- Drag-and-drop or click to upload
- Supports JPEG, PNG, SVG formats
- Maximum file size: 5MB per image
- Multiple images can be uploaded
- Preview appears immediately after upload

**Manage Existing Images**

- View all uploaded images in a grid
- Delete unwanted images with the remove button
- Images are stored permanently until explicitly deleted
- Primary image (first uploaded) displays in flow detail view

#### Asana List Section

**Add Asanas Button**

Located at the top-right of the Asana List section, this button opens a searchable dialog to add poses to your flow.

**How to Add Asanas:**

1. Click "Add Asanas" button
2. Search dialog appears with all available poses
3. Type to search by English or Sanskrit names
4. Click poses to select (checkmarks appear)
5. Click "Add Selected" to insert into flow
6. New poses appear at the end of your sequence

**Asana List Items**

Each pose in your flow displays:

- **Pose Name** - Primary English name with Sanskrit name below
- **Delete Button** - Remove this pose from the flow (trash icon)
- **Move Up/Down Buttons** - Reorder poses in the sequence
- **Alignment Cues Field** - Add custom instructions for this specific pose

**Alignment Cues (Optional)**

- Multi-line text field for each pose
- Maximum 1000 characters per pose
- Add personalized instructions like:
  - "Press through the heels, engage the core"
  - "Breathe deeply for 5 counts"
  - "Focus on hip alignment"
- These cues appear during practice sessions

**Reordering Poses**

Use the up/down arrow buttons to change pose order:

- Up arrow moves the pose earlier in the sequence
- Down arrow moves the pose later in the sequence
- Reorder to create the ideal flow progression
- First pose appears at the top of practice view

**Removing Poses**

Click the trash/delete icon next to any pose to remove it from the flow. There's no confirmation prompt, so be careful when deleting.

### Sticky Bottom Action Bar

The action bar remains visible at the bottom of the screen while editing, providing quick access to save, delete, or cancel operations.

**Save Changes Button**

- Primary action button (blue/primary color)
- Saves all modifications to the flow
- Returns to the flow detail view after saving
- Updates take effect immediately

**Delete Flow Button**

- Destructive action (red/error color)
- Only appears in edit mode (not in create mode)
- Opens confirmation dialog before deletion
- **Warning:** Deletion cannot be undone
- Removes the flow permanently from your library

**Cancel Button**

- Secondary action (outlined)
- Discards all unsaved changes
- Returns to the flow detail view
- No confirmation prompt - changes are lost immediately

## Editing Your First Flow

### Step-by-Step Edit Workflow

1. **Navigate to Practice Flows**

   - Go to Flows â†’ Practice Flows
   - Search for and select the flow you want to edit

2. **Enter Edit Mode**

   - Look for the Edit icon (pencil) next to the flow title
   - Click the icon to transform the detail view into edit mode

3. **Modify Flow Details**

   - Update the flow name if desired
   - Adjust the difficulty level dropdown
   - Revise the description to reflect current practice goals

4. **Add or Remove Poses**

   - Click "Add Asanas" to open the pose selection dialog
   - Search and select new poses to include
   - Click trash icons to remove unwanted poses
   - Use arrow buttons to reorder the sequence

5. **Add Alignment Cues**

   - For each pose, click in the alignment cues field
   - Type detailed instructions for your practice
   - Keep cues concise but descriptive

6. **Manage Flow Images**

   - Upload new images via drag-and-drop or file picker
   - Delete old images that no longer fit the flow
   - Ensure at least one image represents the flow well

7. **Save Your Changes**
   - Review all modifications for accuracy
   - Click "Save Changes" in the bottom action bar
   - Wait for confirmation before navigating away

## Tips for Effective Flow Editing

**Keep Evolution in Mind:**

- Your practice evolves - update flows to match your current level
- Add new poses you've learned recently
- Remove poses that no longer serve your practice

**Maintain Flow Continuity:**

- Ensure pose transitions are smooth and logical
- Start with warm-up poses, progress to peak poses, end with cool-down
- Reorder as needed to maintain natural flow progression

**Use Alignment Cues Wisely:**

- Add cues for challenging poses or personal reminders
- Keep language clear and actionable
- Focus on what helps YOUR practice (not generic instructions)

**Update Difficulty Levels:**

- As you progress, some flows become easier
- Reclassify difficulty to match current challenge level
- Create new variations for different skill levels

**Manage Images Thoughtfully:**

- Use images that inspire and represent the flow essence
- Update images seasonally or when flow purpose changes
- Remove outdated or irrelevant images

## Common Questions

**Q: Can I edit flows created by others?**

A: No, you can only edit flows you created yourself or flows where you have admin permissions. The Edit button only appears for flows you own.

**Q: What happens to my practice history if I edit a flow?**

A: Your practice activity tracking remains intact. Edits to the flow don't affect your historical practice records.

**Q: Can I undo changes after saving?**

A: No, once you click "Save Changes" the modifications are permanent. The Cancel button must be used BEFORE saving if you want to discard changes.

**Q: What happens if I delete a flow I've been practicing?**

A: The flow and all its data are permanently removed. Your practice activity history for that flow is also deleted. This action cannot be undone.

**Q: Can I duplicate a flow instead of editing it?**

A: Currently there's no duplicate feature. To create a variation, use "Create Flow" and manually add the same poses with modifications.

**Q: How many poses should a flow contain?**

A: There's no strict limit, but 5-12 poses work well for most flows. Very short flows (2-3 poses) may feel incomplete, while very long flows (15+) can become unwieldy.

**Q: Do alignment cues appear during practice?**

A: Yes, alignment cues you add for each pose will display when practicing the flow, helping you maintain proper form and focus.

**Q: Can I change the flow creator/owner?**

A: No, the "Created By" field is read-only and cannot be changed. Ownership remains with the original creator.

## Editing Best Practices

**Before Making Major Changes:**

- Consider creating a new flow variation instead
- Note what's working in the current flow
- Have a clear reason for each modification

**When Adding Poses:**

- Ensure new poses fit the flow's theme/purpose
- Consider difficulty progression
- Place new poses in logical sequence positions

**When Removing Poses:**

- Verify the remaining sequence still flows well
- Don't remove poses that serve as bridges between others
- Maintain balance in body parts and pose types

**Alignment Cues Guidelines:**

- Write cues as if teaching yourself
- Focus on your personal alignment challenges
- Include breath reminders where helpful
- Keep under 1000 characters per pose

**Image Management:**

- Upload high-quality, relevant images
- Remove images that no longer represent the flow
- Consider seasonal updates for inspiration

Editing flows is a natural part of your yoga practice evolution. Don't hesitate to refine and improve your flows as your understanding deepens and your practice grows!
